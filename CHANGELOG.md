# Change Log

All notable changes to the "Asset Manage" extension will be documented in this file.

## [0.2.4] - 2026-03-04

#### 中文

- **兼容**：将最低 VS Code 版本要求从 1.90 调整为 1.74

#### English

- **Compatibility**: Adjust the minimum VS Code version requirement from 1.90 to 1.74

## [0.2.3] - 2026-03-01

#### 中文

- **优化**：减小插件打包体积
- **新增**：支持将资源拖拽至工作区编辑器，可快速插入相对路径

#### English

- **Improved**: Reduced extension package size
- **Added**: Support dragging assets to workspace editor for quick relative path insertion

---

## [0.1.0] - 2026-02-17

#### 中文

- **新增**：初始版本发布
- **新增**：静态资源侧边栏功能
- **新增**：图片、媒体、字体、Office 文档预览
- **新增**：字体文件右键预览功能
- **新增**：快捷键支持（Ctrl+Alt+A / Cmd+Alt+A）
- **新增**：资源扫描配置选项

#### English

- **Added**: Initial release
- **Added**: Static asset sidebar
- **Added**: Image, media, font, Office document preview
- **Added**: Font file right-click preview
- **Added**: Shortcut support (Ctrl+Alt+A / Cmd+Alt+A)
- **Added**: Asset scan configuration options
