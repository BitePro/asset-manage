# 告别资源迷宫：Asset Manage —— 让 VS Code 成为你的静态资源管理中心

> 图片散落各处、字体找不到、重复文件占空间？一个插件，全部搞定。

## 你是否有过这些困扰？

- 项目里的图片、字体、音视频散落在不同目录，想找一张图要翻半天

- PDF、Word、Excel 文档想快速看一眼，就得切到系统默认应用

如果你点头了，**Asset Manage** 就是为你准备的。

## 什么是 Asset Manage？

Asset Manage 是一款 VS Code 插件，在侧边栏中为你提供**一站式静态资源管理**。它自动扫描工作区内的图片、音视频、字体、Office 文档等，按类型分类展示，并支持**内置预览**和**重复检测**。

一句话：**把散落的静态资源，收进一个清晰、好用的侧边栏。**

## 核心功能一览

### 📂 分类侧边栏

自动按类型分类：**图片**、**音视频**、**字体**、**办公文档**、**其他**。每个分类下按文件夹分组，一目了然。

### 🔍 大文件标记

自定义体积阈值（如 100KB、1MB），超过阈值的文件会被高亮标记，方便你快速定位「占空间大户」。

### 🔄 重复图片检测

基于内容哈希识别重复图片。点击「查看重复」，可看到所有重复项及所在目录，一键定位，轻松去重。

### 👁️ 内置预览

- **图片**：点击即可预览，支持调节背景亮度
- **PDF**：分页预览，上一页 / 下一页
- **Word(docx)**：在面板内直接渲染
- **Excel(xlsx)**：多工作表切换，表格预览

无需跳出 VS Code，即可完成大部分预览需求。

### 🔤 字体预览

在资源管理器中右键 `.ttf` / `.otf` / `.woff` / `.woff2` 字体文件，选择「字体预览」，侧边栏会自动定位到该字体并展示字形。支持调整字号、展开更多字符集。

### ⌨️ 快捷入口

- 快捷键：`Ctrl+Alt+A`（Windows/Linux）或 `Cmd+Alt+A`（macOS）
- 状态栏快捷入口
- 侧边栏图标

### 🌐 多语言

支持**中文**和 **English**，在侧边栏 Header 中可随时切换。

## 适用场景

- **前端 / 全栈开发**：管理 `assets`、`public`、`static` 等目录下的图片、字体
- **设计协作**：快速预览字体、图片，检查资源是否重复或过大
- **文档项目**：集中浏览 PDF、Word、Excel 等文档
- **媒体项目**：管理音视频文件，支持在线播放预览

## 安装与使用

1. 在 VS Code 扩展市场搜索 **Asset Manage** 并安装
2. 打开任意工作区
3. 点击左侧活动栏的 Asset Manage 图标，或使用 `Ctrl+Alt+A` / `Cmd+Alt+A`
4. 在侧边栏中浏览、搜索、预览资源

配置项（可选）：

- `assetManage.scanInclude`：参与扫描的 glob 列表
- `assetManage.scanExclude`：排除扫描的 glob 列表（默认排除 `node_modules`、`.git`、`dist`、`build` 等）
